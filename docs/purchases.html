Four purchases stand between Adapix today and Adapix reliable, impressive, and professional. Everything is listed in the order it should be bought.

1 · Railway Hobby plan

$5 /month Urgent — ~$2 of trial credit left

The server Adapix runs on, around the clock. When the trial credit hits zero, everything goes offline at once — no follow-ups drafted, no replies answered, dashboard dead. Every other line on this page is pointless if this one lapses.

Where
railway.com/workspace/plans → Hobby → add a card. Includes $5 of usage; the app fits inside it.
Afterward
Nothing to wire — it just keeps running.

2 · Twilio A2P 10DLC registration

~$4 /month + ~$15 one-time

Registers the business with US phone carriers as a legitimate texter. Without it, texts Adapix sends can be silently filtered as spam — the app says "sent," the customer never sees it. The single biggest deliverability fix available.

Where
Twilio Console → Messaging → Regulatory Compliance → US A2P 10DLC. Needs business name, address, and EIN (or sole-proprietor registration).
Afterward
Claude links the sending number to the registered campaign and runs a live deliverability test.
When you're ready for the feature

3 · Blooio iMessage line

$39 /month shared · ~$98 dedicated Code built & tested — activates same day

Blue texts. Follow-ups arrive as real iMessages instead of green SMS from an unknown number — much better open and reply rates on iPhones. Start with the $39 shared line to prove it; upgrade to dedicated when real customers are on it.

Where
blooio.com dashboard → purchase a line. The API key is already configured.
Afterward
Claude fetches the channel ID, wires it to the org, and sends the first blue-text test within the hour.

4 · Dedicated Twilio number + CNAM

~$1.15 /month + small one-time fee

A phone number the business owns, with the company name showing on caller ID. Today calls come from a free number showing just digits; with CNAM the customer's phone says the business name — far higher answer rates, and it lets the website honestly claim "caller ID shows your company" again.

Where
Twilio Console → Phone Numbers → Buy a Number, then Trust Hub → CNAM.
Afterward
Claude imports the number into Vapi with A-level caller attestation and sets it as the org's calling line — the code path already exists.
Free — just needs an account created
